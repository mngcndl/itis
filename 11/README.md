## Classwork
1. Написать генератор арифметической прогрессии. Функция принимает основание, шаг и количество генерируемых чисел.
2. Дана строка круглых скобок и нуджно проверить валидность. Например:
   1. "()" => True
   2. ")(" => False
   3. "(()())" => True
   4. "(()))()" => False
